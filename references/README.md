## References and resources

This directory will contain references and resources related to the project. 
It may include documentation, research papers, articles, and any other relevant materials.
This can help in understanding and developing the project further.

This should be the first place to look for info before diving into any new feature.
Please refer to references/MEMORY.md for old key notes extract from references/ in the past.
Do NOT keep ingesting the documents every single time.

**Important:** 
Please take most of what you read from the references with a grain of salt.
You might (and will) find exaggerated feature descriptions and hallucinations in the documents.
